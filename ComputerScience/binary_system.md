# Binary number system

## 0 과 1

- 컴퓨터는 가장 기본적으로 전자를 사용하여 동작.
  - 전자는 오직 +(1), -(0) 만 가지고 있음.
  - 이러한 이유로 컴퓨터는 **0 과 1 로 이루어진 2진수 시스템**. 즉, Binary number system 으로 동작함.
  
- 어디서 볼 수 있나?
  > **컴퓨터의 RAM 을 확인할 때 128, 256, 512 라는 숫자를 많이 볼 텐데 끝자리를 보면 모두 2로 나누어 떨어진다는 것을 알 수 있음. 표기되는 숫자는 2진수에서 10진수(Deca)로 변환된 것**

- 바이너리 시스템(2진수)는 컴퓨터 공학에서 중요하고 결정적이지만(critical or crucial) 다른 사고 방식이다.

- 작동 방식
  - 우리가 스위치 또는 트랜지스터라고 부르고 있는 것을 끄게 되면 상태는 0이 됨.
  - 컴퓨터 내 단일 CPU 칩에는 0, 1 로 이루어진 바이너리 시스템이 백만개 정도가 있을 수 도 있음.
  - 여러개의 CPU 칩이 0 , 1 을 불규칙적으로 껐다 켰다 하면서 우리에게 기능에 해당하는 서비스를 제공해 줄 수 있음.
  - **`예를 들어`** 이란에 있는 친구에게 메일을 보낸다던가 바탕화면을 업데이트 하든 등등

## 무조건 0 과 1로만 표현할까?

- 우리 인간은 0, 1 로 이루어진 세계보다는 **`Deca System`** 즉, 10 진수로 이루어진 시스템에 익숙하다.
- 10진수는 일관되게 묶여있기 때문에 인간들이 사용하기 쉽다.

### Deca(10진수) 표현

- **10 진수는 0~9 까지 10개의 숫자로 이루어져 있음.**
- 인간에게는 Deca System 이 정말 쉽지만 **컴퓨터는 오직 0, 1 로만 기능을 하기 때문에 우리는 10진수를 2진수로 변환하여 처리해줘야 함.**

### Binary 와 Deca 의 동작 원리는 동일하다

- binary number System 과 Deca number system 은 동일한 원리로 작동함.
- 전혀 다른 유형이지만 말이다.
- 9에서 1이 올라가면 10의 자리에 1이 올라감. 99일때는 100의 자리에 1이 올라가고 모두 00으로 변환
- 2진수는 1 2 4 8 로 증가함.

#### Example 1

- 표로 간단하게 나타낸 10진수 표현방법
  **순서**|100 |10|1|구분|
  :---------:|:---------:|:----------:|:---------:|---------:
  **1**|0 | 0 | 9 | ⬆️
  **2**|0 | 1 | 0 | 😜

  1. 첫번째로 10진수 9를 입력했다고 가정.
  2. 1을 증가 시킴.
  3. 첫번째 1의 자리에는 9 --> 0 으로
  4. 두번째 10의 자리에는 1 을 증가 시킴.

#### Example 2

- 표로 간단하게 나타낸 10진수 표현방법
  **순서**|100 |10|1|구분|
  :---------:|:---------:|:----------:|:---------:|---------:
  **1**|0 | 9 | 9 | ⬆️
  **2**|1 | 0 | 0 | 😜

  1. 첫번째로 10진수 99를 입력했다고 가정.
  2. 1을 증가 시킴.
  3. 첫번째 1의 자리에는 9 --> 0 으로
  4. 두번째 10의 자리에는 1 을 증가 시킴과 동시에 9 + 1 이기에 0 으로 변환
  5. 세번째 100의 자리에 1을 증가 시킴.

#### *16진수도 있는데 16진수는 10진수 끝 9 에 이어 A, B, C, D, E, F 로 이루어져 있음*

## 결론 : 우리는 왜 binary 까지 알아야 하나?

> Computer Science. CS 는 컴퓨터의 대한 연구로 알게 되는 지식들이다. 컴퓨터는 0 과 1로 이루어진 언어를 사용한다 생각한다면 우리는 미국에 여행을 가기 위해 영어를 공부하듯 컴퓨터를 알아가기 위해 2진법, 10진법, 16진법을 공부하는 것이다.  
> 컴퓨터의 연산을 이해하면서 논리를 이해하는 것은 보다 컴퓨터 과학을 공부하는 것에 도움이 될 것이다.
