# 2진수, 10진수, 16진수

- 바이너리 시스템(2진수)는 컴퓨터 공학에서 중요하고 결정적이지만(critical or crucial) 다른 사고 방식이다.

- 컴퓨터에서는 0, 1 을 기반으로 기능을 처리함.
- 우리가 스위치 또는 트랜지스터라고 부르고 있는 것을 끄게 되면 상태는 0이 됨.
- 컴퓨터 내 단일 CPU 칩에는 0, 1 로 이루어진 바이너리 시스템이 백만개 정도가 있을 수 도 있음.
- 여러개의 CPU 칩이 0 , 1 을 불규칙적으로 껐다 켰다 하면서 우리에게 기능에 해당하는 서비스를 제공해 줄 수 있음.
  - 예를 들어, 이란에 있는 친구에게 메일을 보낸다던가 바탕화면을 업데이트 하든 등등

- 우리 인간은 0, 1 로 이루어진 세계보다는 Deca System. 즉, 10 진수로 이루어진 시스템에 익숙하다.
- 10진수는 일관되게 묶여있기 때문에 인간들이 사용하기 쉽다.
- 우리가 10진수 사용할 때 해야할 일은 그저 10 을 계속 곱하는 것 뿐이다.
  - 다른 값을 얻을 때 끝에서 계속 0 을 추가하는 것을 의미.
- 인간에게는 Deca System 이 정말 쉽지만 컴퓨터는 오직 0, 1 로만 기능을 하기 때문에 우리는 10진수를 2진수로 변환하여 처리해줘야 함.
- 10 진수는 0~9 까지 10개의 숫자로 이루어져 있음.

- 16진수도 있는데 16진수는 10진수 끝 9 에 이어 A, B, C, D, E, F 로 이루어져 있음.

- binary number System 과 Deca number system 은 동일한 원리로 작동함.
- 전혀 다른 유형이지만 말이다.
- 9에서 1이 올라가면 10의 자리에 1이 올라감. 99일때는 100의 자리에 1이 올라가고 모두 00으로 변환
- 2진수는 1 2 4 8 로 증가함.
