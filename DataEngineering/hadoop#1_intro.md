# Hadoop #1

## 하둡(Hadoop)이란?

- 정의  
  - 대규모 데이터 처리를 위한 분산 컴퓨팅 프레임워크.
  - 다수의 컴퓨터를 클러스터로 구성하여 대용량 데이터를 분산 처리하고 저장하는데 사용된다.

- HDFS + MapReduce = Hadoop?
  - Hadoop Distributed File System(HDFS)과 분산 데이터 처리를 위한 MapReduce 프로그래밍 모델로 구성
  - HDFS : 대용량 데이터를 클러스터 노드들에 분산하여 저장.
  - MapReduce : 데이터 처리 작업을 분산하여 병렬로 처리.
  - 분산 처리 방식은 대규모 데이터 처리에서 빠른 속도와 확장성.
